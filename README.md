# Exercise-2-SQL-Aggregates-and-Operators
A hands-on SQL practice project focused on aggregate functions and operators. This exercise covers COUNT, SUM, AVG, MIN, MAX, along with GROUP BY, HAVING, and filtering techniques. Includes written SQL queries and expected outputs to strengthen foundational data analytics skills.

This repository contains a structured SQL practice exercise designed to build strong foundational skills in data analytics. It focuses on key aggregate functions such as COUNT, SUM, AVG, MIN, and MAX, as well as essential SQL operations including GROUP BY, HAVING, DISTINCT, and filtering with conditions like BETWEEN and IN.

Each question requires writing SQL queries and predicting the expected output tables, reinforcing both logical thinking and query accuracy. This project demonstrates practical understanding of data grouping, filtering, and summarisation techniques used in real-world data analysis.
